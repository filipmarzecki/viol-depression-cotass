This work is published at https://doi.org/10.1016/j.jad.2025.120005 

Broadly, we investigated the associations between socio-demographic factors, interpersonal violence, and mental health outcomes in Sri Lanka, predominantly reported between 2012 and 2015, in the Colombo Twin and Singleton Study (COTASS) sample, and used genetically-informed methods. 

We used descriptive statistics to describe prevalences of the sample characteristics, as well as the exposures and the outcomes. The script for this can be found in COTASS-Violence-descriptives.Rmd

We used linear regressions with clustered standard errors to investigate the associations between socio-demographic and environmental factors, and experiencing interpersonal violence. The script for this can be found in COTASS-Violence-risks-for-exposure.Rmd

We used linear and logistics regressions with clustered standard errors to investigate the associations between experiencing interpersonal violence and depressive symptoms, depression, and suicidal ideation. The script for this can be found in COTASS-Violence-MH-associations.Rmd

We used the quasi-causal MZ twin differences analysis to investigate the association between interpersonal violence and depression symptoms when accounting for latent familial confounders. The script for this can be found in COTASS-Violence-MZ_diff.Rmd
We used structural equation modelling in OpenMx to quantifying the heritability of depression symptoms and to investigate moderating effect of interpersonal violence on the aetiology of depressive symptoms. The continuous moderation twin model tests whether A, C and E effects increase or decrease as a linear function of a moderator. The extended (bivariate) model allows for the phenotype to be influenced by two sets of variance components – those shared with the moderator, and those that are not. The script for this can be found in COTASS-Violence-bivGxE.Rmd
